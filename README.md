# Melbourne Rental Market Intelligence (2000–2025)

## 📌 Overview
An end-to-end rental market analytics project analysing Melbourne 
suburb-level rental trends from 2000 to 2025. Built to uncover 
pricing patterns, affordability shifts, and demand hotspots across 
Melbourne's diverse suburbs using interactive Power BI dashboards.

## 🎯 Problem Statement
Melbourne's rental market has undergone significant transformation 
over 25 years. This project answers:
- Which suburbs have seen the highest rental growth?
- How has affordability changed across Melbourne regions?
- What are the demand trends by property type and bedroom count?

## 🛠 Tech Stack
- **Data Processing:** Python, Pandas
- **Database:** CSV / Structured Data
- **Visualisation:** Power BI (.pbix)
- **Version Control:** Git, GitHub

## 📊 Dashboard Features
- Suburb-level rental price trends (2000–2025)
- Year-on-year growth analysis
- Regional affordability heatmaps
- Property type breakdown (house, unit, apartment)
